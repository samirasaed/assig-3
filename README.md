MiniShell Project
-----------------
This project implements a simple Unix-like shell using Lex and Yacc.

Features:
- Command execution
- Built-in commands (cd, exit)
- Process creation using fork()
- Program execution using execvp()

Files:
- shell.l : Lexical analyzer
- shell.y : Parser and execution logic
- Makefile : Build automation

How to Run:
make
./minishell
